Release notes

Full history on the GitHub releases page.

v1.0.0

July 2026
  • Mini player — Spotify-style now-playing bar with floating picture-in-picture video; playback survives navigation.
  • Auto-updates via the Tauri updater, system tray with hide-to-tray, magnet-link handling.
  • Loading skeletons, toast notifications, global keyboard shortcuts (Space, F, M, ←/→, Esc).
  • Multi-server Jellyfin login with an upgraded “Who's watching?” profile picker.
  • Torrent search (Prowlarr) + qBittorrent manager with stream-while-downloading.
  • Multi-language UI: English, Español, Français.

v0.9.0 — beta

June 2026
  • First public beta: Netflix-style home, in-app player with subtitles and resume, global search, Docker backend stack.
